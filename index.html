Glossary:

1. Average Calculator: A tool designed to calculate the average value of a set of numerical inputs. In the context of the provided calculator, it computes the average grade of five subjects based on the grades entered by the user.

3. Input Field: A space on a web page where users can enter data or information. In the Average Calculator, input fields are provided for entering the grades of the five subjects.

4. Calculate Button: A button on the web page that, when clicked, triggers the calculation process. In the calculator, the "Calculate Average" button computes the average grade based on the inputted values.

5. Average Grade: The result obtained by summing the grades of multiple subjects and dividing by the total number of subjects. The Average Calculator provides the average grade of the entered subject grades.

6. Mute Button: A button that allows users to silence or unmute the background sound. In the context of the calculator, the mute button enables users to control the background music with a simple click.

7. User Experience (UX): The overall experience of a person using a product or service, such as a website or application. The Average Calculator aims to provide a positive user experience by offering a straightforward interface for calculating grades.

8. Web Browser: A software application used to access information on the World Wide Web. Users interact with web pages, like the Average Calculator, through a web browser to input data and view results.

9. Text Editor: A software program used for creating and editing plain text files. In the context of the calculator, a text editor is used to save the HTML code and create the web page for the calculator.

10. Responsive Design: A design approach that ensures web pages adapt and display appropriately on various devices and screen sizes. The Average Calculator is designed to be responsive for optimal viewing on different devices.
